# Monster Index

One file per monster. Each file contains a flavor description and a custom-ruleset stat block (Might / Finesse / Wit / Presence + Evasion + traits/actions).

Stats: Might (STR), Finesse (DEX), Wit (max of INT/WIS), Presence (CHA). Evasion replaces AC. Attack rolls use the relevant stat modifier only (no proficiency bonus). Saves are passive (10 + stat modifier) unless a special bonus is listed.
