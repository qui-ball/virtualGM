# Class Abilities

Starting and level-up abilities are split by class:

- [Warrior](warrior.md)
- [Mage](mage.md)
- [Ranger](ranger.md)
- [Bard](bard.md)

At level 1, each character chooses **one** starting ability. At levels 2-10,
when selecting "class ability" for level-up, pick one option from the class
table, subject to level restrictions.
