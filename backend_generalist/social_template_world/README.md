# Social Session World Directory

Loose social-play seed for testing the generalist GM.

## Layout
- `pc.json` - PC-facing resources, reputation, favors, and notes.
- `campaign/` - Broad scenario reference. Read only what is useful.
- `world/scene.json` - Current place, tone, and who is present.
- `world/encounter.json` - Optional tracker for an active tense exchange.
- `world/npcs.json` - Lightweight NPC attitudes and remembered facts.
- `rules/core-ruleset.md` - Core custom system rules.
- `rules/spell-list.md` - Mage and Bard spell details.
- `rules/class-abilities.md` - Warrior, Mage, Ranger, and Bard ability tables.

## Intent
Keep the fiction open. Use these files as starting pressure, not a script.
NPCs should have motives and limits, but their exact responses should emerge
from the player's approach and the current state.
NPC names in reference files are private labels until introduced in the fiction.
