## Redbrand Hideout

**Level:** 2 · **Location:** `loc-redbrand-hideout` (under Tresendar Manor)

### Entry

- **Manor basement** (locked doors, Redbrand guards), or
- **Secret tunnel** via Carp Alderleaf’s farm (Qelline’s son) — quieter approach.

### Key areas (abstract)

| Area | Content |
|------|---------|
| Barracks | Redbrand ruffians sleeping or gambling |
| Trapped hall | Optional pit or crossbow trap |
| Nothic pit | Optional **nothic** (`enemy-nothic`) — psychic hunger, bargains for secrets |
| Prisoners | Locals held for ransom — rescue improves town support |
| Glasstaff’s lab | **Iarno Albrek** (`npc-glasstaff`, `enemy-glasstaff`) — mage with staff |

### Glasstaff / Iarno

- Poses as town ally; true Redbrand leader.
- Carries correspondence mentioning **the Black Spider** and interest in **Wave Echo Cave**.
- If losing: casts invisibility / misty escape unless cornered — may become recurring threat if fled.
- If captured: townsfolk celebrate; Sildar wants him held for trial.

### Aftermath

- Phandalin safer; townsfolk share rumors about goblins, dwarves, and “a spider in the shadows.”
- **Sildar** and/or **Garaele** point toward wider region quests → Part 3.

### Story elements

- **ev2-hideout-clear** — Prisoners freed; gang broken.
- **ev2-glasstaff** — Iarno confronted; Black Spider link revealed.
- **tr3-spider-web** — Regional quests. *Load:* `Part3_The_Spiders_Web/Overview`.

### XP

**75 XP** for clearing hideout; **+50 XP** if Glasstaff defeated or captured.
