## Cragmaw Castle

**Level:** 3 · **Location:** `loc-cragmaw-castle`

### Overview

Ruined castle in Neverwinter Wood. **King Grol** (`npc-grol`, `enemy-grol`) rules Cragmaw goblins and **hobgoblins** (`enemy-hobgoblin`). **Gundren Rockseeker** (`npc-gundren`) is prisoner. A **doppelganger** agent of the Black Spider may pose as Gundren to steal the map.

### Key areas (abstract)

| Area | Content |
|------|---------|
| Outer walls | Hobgoblin patrols |
| Courtyard | Goblin squabbles; wolf pens optional |
| Throne room | King Grol, Gundren (or doppelganger), bodyguards |
| Postern / escape | Goblins flee if Grol falls |

### King Grol

- Arrogant; may parley if PC offers leverage.
- Knows Gundren goes to **Nezznar** (the Black Spider) when “the map” is recovered.
- Loot: crown trinkets, food stores, **Gundren’s map** if not already taken.

### Gundren

- Weak but alive if rescued in time.
- If doppelganger revealed: fight or chase — map must be recovered.
- **Map** shows entrance to **Wave Echo Cave** — transition to Part 4.

### Nezznar hint

Grol or captured goblins name the villain: **Nezznar**, a drow, commands from the mine.

### Story elements

- **ev3-grol** — Confrontation with King Grol.
- **ev3-gundren-rescue** — Gundren saved; map obtained.
- **ev3-black-spider-name** — Nezznar identified as Black Spider.
- **tr4-mine** — Load `Part3_The_Spiders_Web/Whats_Next` then Part 4.

### XP

**100 XP** milestone for castle cleared and Gundren rescued.
